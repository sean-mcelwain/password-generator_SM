# Random Password Generator - Read ME
<a name='description'></a>
## Description

This project is a webpage containing a Password Generation Utility that allows a user to generate a random password.  The utility creates a password of a specified length (between 8 characters and 128 characters) that can contain UPPERCASE letters, lowercase letters, Numbers, and/or Special Characters. This app utilizes: (1) HTML, (2) CSS, and (3) Javascript.
